<h2>GPU Alert</h2>

A Python application written in OOP style to send email alerts for GPU stock availability during the 2020-21 shortage.

Names of retailers in code and in configuration files have been removed to protect their rights. This project is intended as a reference only.

Made using AWS SES, BeautifulSoup4 and requests.
